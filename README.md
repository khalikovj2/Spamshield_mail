# SpamShield — Email Spam Classifier

A clean, minimal web app that detects spam emails instantly using TF-IDF + Logistic Regression.

**Live Demo:** Deploy on GitHub Pages or Vercel

## Features
- Instant spam / ham classification
- Confidence score + probability bars
- Spam indicator detection
- Scan history
- Works 100% offline — no backend needed

## How to run
Just open `index.html` in your browser. No installation needed.

## Deploy on GitHub Pages
1. Push this repo to GitHub
2. Settings → Pages → Source: main branch
3. Done ✅

## Tech
- Pure HTML, CSS, JavaScript
- TF-IDF feature extraction
- Logistic Regression classifier
- Zero dependencies
